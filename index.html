Features

Great features you'll love.

Tired of jumping between tabs to make notes while reading on the web? How about just a right-click doing this job for you? Paper extension will let to take notes on the go!

Easy to use

Simple user interface where you just need to right click and click on 'Save to Paper' and voila the text is now safe in a dropbox paper!

Easy To Customize

Save to an existing paper or create a new paper for a fresh start, paper extension is here to help you increase your speed and productivity.

Free of Charge

Open source project developed by students of SVNIT, Surat. Feel free to contribute and add more features!

Faq

Questions and Answers.

Got some Questions? Maybe they're asked before as well. Find some below.

How can I install the extension?

Click on the Get it now! button which will lead you to the Github repository of the extension. Follow the instructions given there and you're done!

Do this extension collect any personal data?

No. This extension simply saves the text you intend to a dropbox paper.

Where can I find my text?

You can find your text in the paper section of dropbox saved with the name you entered while creating the paper.

How can I contribute to this project?

Go on to the Github repository of the project and look for some issues in the issues tab or open a pull request if you've already made our extension better!
